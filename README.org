#+TITLE: Hit

Hit is a library with a command line tool of a full text search implementation.
It provides an index structure that allows to add, search and delete documents.
It can only process text documents.

It is a simple implementation that I started to learn [[https://ocaml.org/][OCaml]]. It does not provide
stemming and stop words.

The index is stored in a storage backend that can be implemented for different
storage systems. A file system storage implementation is provided.

* Library

To use the library in your own dune project, add the following to your ~dune~ file:
#+begin_src dune
  (libraries hit)
#+end_src

** Initialize Index

First you have to create the index with a given path:

#+begin_src ocaml :exports both :results pp :tangle example.index-api.utop
  #require "hit";;
  open Hit
  let index = Index.create "./my_index";;
#+end_src

#+RESULTS:
: val index : Hit.Index.t = {Index.path = "./my_index"}

** Add Documents

Add a document to the index:

#+begin_src ocaml :exports both :results pp :tangle example.index-api.utop
  [
    Index.add index "/documents/doc1.txt" "This is the content of document one.";
    Index.add index "/documents/doc2.txt" "This is the content of document two.";
    Index.add index "/documents/doc3.txt" "This is the content of document three.";
    Index.add index "/documents/doc4.txt" "This is the very long content
  					    Lorem ipsum dolor sit amet, consetetur sadipscing elitr,
  					    sed diam nonumy eirmod tempor invidunt ut labore ets
  					    dolore magna aliquyam erat, sed diam voluptua
  					    of document four.";
  ] |> List.map Doc.Id.to_string;;     
#+end_src

#+RESULTS:
: - : string list =
: ["doc-a9741b4cd15ed6a57b57a9c945440e3f";
:  "doc-c92b8f628082cb9f983e42c4becb450c";
:  "doc-cb336f373ba3e37f32c939c484699259";
:  "doc-c42f3d8d7d68356c3922656194d30a52"]

~add~ parses the content and updates the index.
If you add a document with the same source and path again, the document
will be updated in the index if the content is different.
For example, let's update document two:

#+begin_src ocaml :exports both :results pp :tangle example.index-api.utop
  [
    Index.add index "/documents/doc1.txt" "This is the content of document one.";
    Index.add index "/documents/doc2.txt" "This is the content of document II. Will be updated!"
  ] |> List.map Doc.Id.to_string;;     
#+end_src

#+RESULTS:
: - : string list =
: ["doc-a9741b4cd15ed6a57b57a9c945440e3f";
:  "doc-c92b8f628082cb9f983e42c4becb450c"]

** Search Documents

There is a query language that you can use to search documents:

#+begin_src ocaml :exports both :results pp :tangle example.index-api.utop
  let results = Index.query index "(or (eq one) (eq three))";;
  assert ((List.length results) = 2);;
  List.map Doc.Id.to_string results;;
#+end_src

#+RESULTS:
: - : string list =
: ["doc-a9741b4cd15ed6a57b57a9c945440e3f";
:  "doc-cb336f373ba3e37f32c939c484699259"]

** Delete documents

It is also possible to delete documents by their id:

#+begin_src ocaml :exports both :results silent :tangle example.index-api.utop
  Index.delete index (Doc.Id.from_string "doc-c92b8f628082cb9f983e42c4becb450c")
#+end_src

* CLI

** Install

To install the command line tool, copy the executable in your bin directory.
If you have ~$HOME/.opam/default/bin/hit~ in your path, simply run this:

#+begin_src bash
  make install
#+end_src

** Usage

Let's add some documents:

#+begin_src bash :exports both :results pp
  hit add -d my_index README.org
  hit add -d my_index LICENSE
#+end_src

#+RESULTS:
: doc-aacb4ce123f9200898dff4a35e6a8461
: doc-9879d6db96fd29134fc802214163b95a

Let's search for the documents:

#+begin_src bash :exports both :results pp 
  hit query -d my_index "(or (eq test) (eq shall))"
#+end_src

#+RESULTS:
: LICENSE
: README.org

Let's delete a document by its id:

#+begin_src bash :exports both :results pp 
  hit delete -d my_index doc-0e6ae644a708fec47caa0fb606f4fa38
#+end_src
